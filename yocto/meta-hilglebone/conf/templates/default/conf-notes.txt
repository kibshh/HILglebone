### HILglebone Yocto build environment ready.

Common targets:

    bitbake hilglebone-image              # build the full target image
    bitbake -p hilglebone-image           # parse only (sanity check, no build)
    bitbake -c menuconfig virtual/kernel  # tweak the kernel config

Output after a successful build:

    tmp/deploy/images/beaglebone/hilglebone-image-beaglebone.wic.xz

To flash:

    sudo bmaptool copy tmp/deploy/images/beaglebone/hilglebone-image-beaglebone.wic.xz /dev/sdX
